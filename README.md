## 机械动力：懒惰刻 ⚙️

---

一个基于懒惰刻的极致优化模组，可以减少机械动力工作元件的巨量工作占用和常态占用🔄，可以让服务器相比于纯机械动力多带2-3倍机械动力产线！

1. **传送带** 减少传送带静态占用, 常态化下传送带相比于原版削减 2-4倍占用。
2. **通过缓存机制，巨量削减动力合成器、动力搅拌器、动力锯在大量配方下的占用（常态占用低至2us！原版可达300us）**
3. **漏斗** 漏斗静态占用相比于原版削减 5-7 倍占用。
4. **溜槽** 降低溜槽的实体刻，相比于原版削减 2-3 倍占用。
5. **分液池** 降低实体刻，相比于原版削减 10倍异常占用，1.2 倍常态占用。
6. **动力锯** 降低实体刻，相比于原版削减 30倍异常占用，10 倍常态占用。
7. **置物台** 降低实体刻，相比于原版削减 8 倍常态占用。
8. **流体系统** 降低流体刻，相比于原版削减 4 倍常态流体占用（可以在配置文件内修改调整倍）。
9. **动力合成器**
10. **机械臂**
11. **机械手**
12. **动力搅拌器**

**注意**: 懒惰刻会导致一些原件动画出现略微的延迟，这是不可避免的，可通过修改配置文件来找到最合适的参数。
